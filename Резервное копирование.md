# Задание 1: Команда rsync с хэш-суммами
Цель: Создать зеркальную копию домашней директории пользователя в директорию /tmp/backup с исключением скрытых файлов и директорий. Использовать подсчёт хэш-сумм для сравнения файлов.
## 1.1. Команда rsync
Разработанная команда для выполнения задания:
`rsync -avh --checksum --delete --exclude='.*/' --exclude='.*' ~/test_backup/ /tmp/backup_test/`

Описание параметров команды:

• -a (archive mode) — режим архивирования, сохраняет права доступа, владельца, время модификации и символические ссылки

• -v (verbose) — подробный вывод операций, показывает какие файлы копируются

• -h (human-readable) — человекочитаемый формат размеров файлов (KB, MB, GB)

• --checksum — сравнение файлов по контрольной сумме (MD5/SHA), а не по размеру и времени модификации. Это гарантирует обнаружение изменений даже если размер и дата не изменились

• --delete — удаление файлов в приёмнике, которых нет в источнике (полностью зеркальная копия)

• --exclude='.*/' — исключить все скрытые директории (начинающиеся с точки)

• --exclude='.*' — исключить все скрытые файлы (начинающиеся с точки)
